# Paper

This folder contains the manuscript source and compiled PDF for:

**A10 Claim-Bounded Black-Hole Interior Re-Exploration Archive: Synthetic Diagnostic Candidate Mapping and External-Comparator Handoff under the A10 Core Methodology**

The manuscript is intentionally framed as a claim-bounded diagnostic archive and external-comparator handoff report. It does not claim black-hole interior discovery, QES detection, singularity resolution, beyond-GR evidence, numerical-relativity validation, or accepted physical candidates.

The title page includes a blank GitHub Repository URL field below the date so the repository URL can be inserted later by the author.
